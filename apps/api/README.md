# Site Yonetim API (Sprint 1 - FastAPI Baslangic)

Bu klasor Sprint 1 kapsaminda FastAPI backend iskeletini icerir.

## Icerik
- App factory + versioned API (`/api/v1`)
- Health endpoint (`GET /api/v1/health`)
- Pydantic settings (`.env` uyumlu)
- SQLAlchemy base + ortak mixinler (timestamp, soft delete, tenant)
- Alembic migration iskeleti
- Pytest smoke testi

## Hızlı Başlangıç
1. Bagimliliklari kur
2. API'yi calistir
3. Testleri kos

Detayli komutlar monorepo root README'sine Sprint 1 sonunda eklenecek.
